reverend
========

Kernel Bayes' Rule implementation and extensions as part of my PhD.


Requirements
------------

For the C++ Executables:
* Boost 1.53 (may work with earlier versions)
* Eigen3
* Nlopt
* cnpy (https://github.com/rogersce/cnpy)

For the Python scripts:
* Python2
* Numpy/Scipy
* Matplotlib
